# PopMath

A Python-based utility designed to break down retail beverage pricing and combat confusing product sizing.

### Purpose
PopMath empowers the user to demystify volume-versus-price schemes at the grocery store, making it easy to calculate the true cost per ounce of soda. Whether comparing individual cans, 2-liter bottles, or bulk multi-packs, this tool strips away marketing layers to find the absolute best value.

### Features
* **Ounce-for-Ounce Comparison:** Instantly translates total package fluid ounces against price points.
* **Smart Sizing Defense:** Bypasses misleading retail deals to expose the real unit cost.
* **Mobile Deployment:** Configured with Buildozer to run seamlessly as an Android application right in the store aisle.
